# Tables


Zulip supports Markdown formatting for tables.




## Examples


### What you type


wzxhzdk:0


!!! tip ""

    The initial pipes (`|`) are optional if every entry in the first column is
    non-empty.
    The header separators (`---`) must be at least three dashes long.
